# How to Get Your Public Production URL

## The URL you have is a PREVIEW deployment (requires login)
`focus-wall2-p7akiejio-inna-berkovichs-projects.vercel.app`

## You need the PRODUCTION URL (no login required)

### Option 1: Check Vercel Dashboard
1. Go to https://vercel.com/dashboard
2. Click on your `focus-wall2` project
3. Look at the top - you'll see the **Production** domain
4. It should be: `https://focus-wall2.vercel.app` (or similar, without the hash)

### Option 2: Promote Current Deployment to Production
1. Go to https://vercel.com/dashboard
2. Click on `focus-wall2` project
3. Go to the **Deployments** tab
4. Find your latest deployment
5. Click the **three dots** (⋯) menu
6. Click **"Promote to Production"**

### Option 3: Check Project Settings
1. Go to your project → **Settings** → **Domains**
2. The production domain will be listed there
3. It should be: `focus-wall2.vercel.app`

## After you have the production URL:
- Share: `https://focus-wall2.vercel.app` (or whatever your production domain is)
- This URL will work WITHOUT login for anyone
- Preview URLs (with hash) require Vercel authentication
